# Тестирование методов BooksCollector

## Список тестов:

- **test_add_new_book_expected_count** проверка ожидаемого количества добавленных книг
- **test_set_book_genre** -- проверка установки жанра для книги
- **test_get_book_genre_by_book_name** -- проверка получения жанра по названию
- **test_get_books_with_specific_genre** -- проверка получения книги по определённому жанру
- **test_get_books_genre** -- проверка получения словаря жанров
- **test_get_books_for_children** -- проверка получения книг, подходящих детям
- **test_add_book_in_favorites** -- проверка добавления книги в Избранное
- **test_add_book_in_favorites_not_in_genre** -- проверка добавления в Избранное книги не из списка
- **test_delete_book_from_favorites** -- проверка удаления книги из Избранного
- **test_get_list_of_favorites_books** -- проверка получения списка книги из Избранного
